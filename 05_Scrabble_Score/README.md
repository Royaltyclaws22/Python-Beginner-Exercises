# 🧩 Scrabble Score

A simple Python program that calculates Scrabble scores for English words.

## ⚙️ Features

- Calculates the Scrabble score of English words.
- Supports case-insensitive word input.
- Reads multiple words from an external text file.
- Calculates the total Scrabble score of all words.
- Finds the word with the highest Scrabble score.
- Counts the number of words with exactly 29 points.


## 💻 Installation & Running

1. Clone the repository:
```bash
git clone https://github.com/Royaltyclaws22/Python-Beginner-Exercises.git
```
2. Navigate to the project folder:
```bash
cd Python-Beginner-Exercises/05_Scrabble_Score
```
3. Run the program:
```bash
python Scrabble_Score.py
```

Make sure that the input file `english_words.txt` is located in the same directory as `Scrabble_Score.py`.


## 💡 Usage

After running the program, it will:

1. Calculate the Scrabble score of all alphabet letters as a test.
2. Read the words from `english_words.txt`.
3. Display:
   - The total Scrabble score of all words.
   - The word with the highest Scrabble score.
   - The number of words with exactly 29 points.

You can modify the example values in the code and the content of the input text file according to your needs.
