# 🔢 Roman Numerals

A simple Python program that converts integers to Roman numerals and Roman numerals back to integers, while also processing input files with numerical data.

## ⚙️ Features

- Convert integer numbers to Roman numeral format.
- Convert Roman numerals to decimal numbers.
- Count Roman numeral conversions with more than 7 characters.
- Count Roman numerals representing numbers divisible by 7.
- Reads data from input text files.


## 💻 Installation & Running

1. Clone the repository:
```bash
git clone https://github.com/Royaltyclaws22/Python-Beginner-Exercises.git
```
2. Navigate to the project folder:
```bash
cd Python-Beginner-Exercises/04_Roman_Numerals
```
3. Run the program:
```bash
python Roman_Numerals.py
```

Make sure that the input files `integers.txt` and `roman_numerals.txt` are located in the same directory as `Roman_Numerals.py`.


## 💡 Usage

The program automatically:

1. Converts example values between decimal numbers and Roman numerals.
2. Reads integers from `integers.txt` and counts those whose Roman representation has more than 7 characters.
3. Reads Roman numerals from `roman_numerals.txt` and counts those representing numbers divisible by 7.

The results are displayed in the terminal.

You can modify the example values in the code and the contents of the input text files according to your needs.
