# ✂️ Rock, Paper, Scissors

A simple Python command-line Rock, Paper, Scissors game where users can compete against the computer, which makes a random choice each round, while keeping track of the final score.

## ⚙️ Features

- Play against a computer opponent.
- Random computer choices.
- Detects wins, losses, and draws.
- Tracks the number of player and computer wins.
- Accepts user input regardless of letter case.
- Quit the game at any time by typing `Q`.


## 💻 Installation & Running

1. Clone the repository:
```bash
git clone https://github.com/Royaltyclaws22/Python-Beginner-Exercises.git
```
2. Navigate to the project folder:
```bash
cd Python-Beginner-Exercises/01_Rock_Paper_Scissors
```
3. Run the program:
```bash
python Rock_Paper_Scissors.py
```


## 💡 Usage

- Type `rock`, `paper`, or `scissors` and press `Enter`.
- Type `q` to quit the game.
- The final score is displayed when the game ends.
